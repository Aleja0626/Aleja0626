### Projects

**Ticket Classifier** -- AI-powered multi-language ticket routing system for Zoho Desk. 11 versions, 67 nodes, classifies by category, priority, and department with business-rule overrides. Handles Spanish, Catalan, and English.

**GO2CONEXIA** -- Chrome extension (Manifest V3) for automated attendance tracking. Enforces clock-in before accessing work tools, with n8n backend for shift management and auto-checkout.

**ConexiaBot** -- Internal support chatbot on Zoho Cliq backed by a vector knowledge base. Agents ask questions in natural language and get sourced answers from company documentation.

**WhatsApp Business Integration** -- Conversational bot with multi-turn flows (greeting, problem capture, callback scheduling) plus round-robin agent assignment in Zoho Desk.

**Centralized Token Manager** -- Single n8n workflow that handles OAuth refresh for all Zoho credentials, consumed by every other workflow via Execute Workflow calls.
